# E-Commerce Sales Dashboard

## Project Overview

This project analyzes e-commerce sales data using Python, SQL, and Power BI to identify business trends and generate actionable insights.

## Technologies Used

* Python
* SQL
* Power BI
* Pandas
* NumPy

## Features

* Sales performance analysis
* Customer analysis
* Return rate tracking
* Delivery rate monitoring
* Interactive dashboard creation

## Key Performance Indicators (KPIs)

* Total Orders
* Total Customers
* Return Rate
* Delivery Rate
* Revenue Trends

## Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. Data Analysis using SQL
4. Dashboard Creation in Power BI
5. Insight Generation

## Outcome

* Identified customer and sales trends
* Built interactive visualizations
* Generated business insights for decision-making
